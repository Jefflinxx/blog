# User Manual

This page is a place to explain how to work with me effectively.

## What I Value

- Clear communication
- Fast iteration
- Thoughtful ownership
- Simple systems

## How to Communicate

Write directly. Lead with the point, then add context when it changes the decision.

## Working Together

The best collaboration is specific, honest, and easy to act on.
